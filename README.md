# 🚀 Career Writer

Career Writer is an AI-powered web application that helps users generate professional resumes and formal emails effortlessly. Built with **Python**, **Streamlit**, and the **OpenAI API**, the application leverages Large Language Models (LLMs) to create high-quality, ATS-friendly resumes and polished professional emails. Users can also export the generated content as PDF documents for easy sharing and printing.

---

## ✨ Features

- 🤖 AI-powered Resume Generator
- 📧 Professional Email Generator
- 📄 Export generated content as PDF
- ⚡ Fast and responsive Streamlit interface
- 🎯 ATS-friendly resume generation
- 🖥️ Simple and user-friendly UI

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Career-Writer.git
```

### 2. Navigate to the project folder

```bash
cd Career-Writer
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Create a `.env` file

```env
OPENAI_API_KEY=your_api_key_here
```

### 5. Run the application

```bash
streamlit run career_writer_app.py
```
